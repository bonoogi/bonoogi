### 운동한컷(iOS)

'운동한컷'은 HealthKit으로부터 사용자의 아이폰에 기록된 운동 기록을 받아와 사진 위에 그려서 공유할 수 있도록 해주는 앱입니다. 사실 비슷한 앱이 있으면 안만들려고 했는데 없어서 그냥 제가 만들었습니다.
